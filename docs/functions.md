# Функциональные модели 
## Демнострация меню
- Пользователь открывает приложение.
- Приложение отображает меню игры.
- Пользователь нажимает на одну из нескольких кнопок.
- Приложение обрабатывает сценарий в зависимости от нажатой кнопки.
